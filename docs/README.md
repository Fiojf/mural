# Mural — marketing site

Static landing page for [Mural](https://github.com/Fiojf/mural). Plain HTML, CSS, vanilla JS — no build step. Edit files, commit, push.

## Deploy

To enable GitHub Pages from this folder: open the repo on GitHub, go to **Settings → Pages → Source**, choose **Deploy from a branch**, set **Branch: `main`** and **Folder: `/docs`**, then **Save**. The site publishes to `https://fiojf.github.io/mural/` within a minute or two.

`.nojekyll` and `_config.yml` (with `theme: null`) are present so Jekyll does not strip files or apply a default theme.
