Features

Built for the way you work.

A wallpaper picker that respects your flow. Summon, switch, dismiss — all without leaving what you were doing.

  • Summon from anywhere

    A configurable global hotkey opens Mural over any window, on any Space. The popover floats above everything and dismisses when you click away — no app to switch to, no dock icon to wrangle.

  • Three layouts, one panel

    Horizontal scroller for quick flips, grid for browsing, vertical list for a keyboard-driven jump. The popover resizes to fit the layout you pick — 820×240, 720×480, or 380×560.

  • Forty-six themes built in

    Catppuccin, Tokyo Night, Rose Pine, Gruvbox, Nord, Everforest, Ayu, Dracula and friends — ready out of the box. Drop a TOML into your config folder to ship your own.

  • Wallpapers from any GitHub repo

    Point Mural at a public repository and it shallow-clones the images locally, then re-syncs on a schedule you control. A cheap ls-remote check skips work when nothing has changed.

  • Per-display targeting

    Set a different wallpaper on each monitor without leaving the picker. Mural matches displays by their stable NSScreenNumber, so the right wall stays on the right screen.

  • Auto-rotate

    Cycle wallpapers every N minutes, or sync to local sunrise and sunset via CoreLocation. Pause it whenever; it picks up where it left off.

Sources

Wallpapers from anywhere.

Drop images in ~/Mural and Mural watches the folder for new files. Need a curated set you don't want to manage by hand? Point Mural at a public GitHub repo and it shallow-clones the images for you, syncing on a schedule.

  • Local folder watcher via notify
  • Multiple GitHub sources, each with its own ref & subpath
  • Per-source sync interval, on-demand refresh
  • Disambiguates duplicate filenames across sources
config.toml 
# Pull from a curated wallpaper repo
[[sources]]
kind = "github"
url  = "https://github.com/dharmx/walls"
ref  = "main"
path = "macOS"
sync_interval_hours = 24

# Plus your own folder
[[sources]]
kind = "local"
path = "~/Mural"

How it works

Three steps. Then forget it's there.

  1. 01

    Drop wallpapers

    Save images to ~/Mural, or add a public GitHub repo as a source.

  2. 02

    Press the hotkey

    ⌘⇧W summons the floating picker, anywhere on macOS.

  3. 03

    Click and you're done

    The wallpaper applies instantly. No menus, no dialogs, no confirmation.
